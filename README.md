☕ Cafeteria Aurora

Landing Page desenvolvida como projeto de estudos para praticar HTML5 e CSS3.

  Sobre o projeto

A Cafeteria Aurora é uma cafeteria fictícia criada para colocar em prática os conhecimentos adquiridos durante meus estudos em desenvolvimento Front-End.

Neste projeto, busquei aplicar boas práticas de estruturação HTML, organização de CSS e construção de um layout moderno e agradável.

 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Google Fonts

 Funcionalidades

- Landing Page responsiva
- Menu de navegação
- Seção "Nossa História"
- Cardápio
- Contato
- Design moderno

 Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar meus conhecimentos em HTML e CSS, praticando conceitos como:

- Estrutura semântica
- Flexbox
- Variáveis CSS
- Responsividade
- Organização de código
